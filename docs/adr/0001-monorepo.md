# ADR-0001: 프론트엔드와 백엔드 모노레포 사용

- 상태: Accepted
- 결정일: 2026-07-15

## 배경

Vue 프론트엔드와 Spring Boot 백엔드의 API 계약, 검증 설정, 배포 문서를 한 저장소에서 함께 관리해야 한다.

## 결정

루트 아래 `frontend/`와 `backend/`를 분리한 모노레포를 사용한다. 공통 하네스와 CI는 루트에 두고, 영역별 규칙은 각 폴더의 `AGENTS.md`에 둔다.

## 결과

- 하나의 MR에서 API와 소비자 변경을 함께 검토할 수 있다.
- CI는 프론트엔드와 백엔드를 독립 단계로 실행한다.
- 배포 결과물은 프론트 정적 파일과 백엔드 Docker 이미지로 분리한다.
